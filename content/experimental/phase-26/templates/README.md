# Phase 26 Proof-Skeleton Templates

These files are private proof-skeleton templates for Phase 26 candidate
authoring. They describe intended proof pressure, preferred proof signals, and
rejection traps.

They are not automatically player-facing content.

They are also not guaranteed `pnpm authoring -- sample` input contracts. The
current sample command requires concrete `guestCount` and `rules`; these
templates are scaffolds that must be converted into candidate Puzzle Schema JSON
files under `content/experimental/phase-26/candidates/` before they count as
serious attempts.

See `docs/phase-26/authoring-workflow.md` for the full candidate loop.
